#  Gemini CV Analyzer

An AI-powered CV analysis application built with Streamlit and Google Gemini Vision.  
Upload a résumé image, ask any question about its content, and the model extracts key details, evaluates the profile, or generates tailored insights.

---

##  Features

- Upload any résumé image (PNG, JPG, etc.)
- Extract name, experience, education, skills, and more
- Ask free-form questions about the CV content
- Powered by **Gemini 2.5 Flash** for fast multimodal inference
- Simple, user-friendly Streamlit interface

---

##  Tech Stack

- **Python 3.x**
- **Google Gemini** (`google-generativeai`)
- **Streamlit**
- **Pillow (PIL)**
- **python-dotenv**
